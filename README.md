# ☀️ Solar Power Trend Analysis

## 📌 Overview
This project analyzes solar power generation using data science techniques such as:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Visualization
- Machine Learning

---

## 📊 Key Insights
- Solar power follows a daily cycle
- Peak generation occurs between 11 AM – 2 PM
- No power generation at night
- Strong correlation between AC and DC power
